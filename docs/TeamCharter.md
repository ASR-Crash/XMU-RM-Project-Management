# 团队章程

## 第一章 总章程

### 团队名称：

		RCS

>		R:Robot（机器人）
>		C:Creative（创新）Contributions（奉献）Cooperation（合作）
>		S:Science（科学） Stick（坚持）

### 团队性质：

		多学院协作、多专业融合的机器人创新团队

### 团队目标：

		RM2022赛季：全国16强
		RM2023赛季：全国8强

### 发展方向：

		成长为肯钻研、能创新、有实力的机器人强队

### 团队追求：

		刻苦钻研，技术至上

#### <a name="1">**考勤制度**</a>
----
考勤范围：

		例行出勤
		每周例会
		团队建设
		官方直播

考勤文件：

> 厦门大学 RM2021 例行出勤考勤······等4个文件

例行出勤：

		周六/寒假/暑假（9小时） 
		08：30~11：30 
		14：30~17：30 
		18：30~21：30
		
		周天（3小时）
		08：30~11：30


		法定节假日根据实际情况安排

非例行出勤：

		建议每周累计工时不少于6小时

每周出勤时间：

		9+6+3=18小时（2个工作日）

考勤影响：

		梯队队员转为正式队员
		正式队员团队排名认定
----
#### <a name="2">**考核制度**</a>
----

新队员->梯队队员考核：

		校内赛表现（50%）
		队内评价（30%）
		考勤结果（20%）
梯队队员->正式队员考核：

		任务完成情况（50%）
		队员评价（30%）
		考勤结果（20%）

#### <a name="3">**会议制度**</a>

----

会议类型：

		全体会议
		每周例会
		管理层会议
		技术组会议
		兵种组会议
		运营组会议

会议对象：

> 全体会议

		全体队员，指导老师，顾问

> 每周例会

		队长，副队长，项目管理，技术组，（运营组）（指导老师）（顾问）

> 管理层会议

		队长，副队长，项目管理，（指导老师）（顾问）

> 技术组会议

		机械组/电控组/视觉组，（指导老师）（顾问）

> 兵种组会议

		英雄组/工程组/步兵组/无人机组/哨兵组/飞镖组/雷达组

> 运营组会议

		宣传经理，招商经理，运营组组员，（队长）（指导老师）

每周例会时间：

		周六早上08：30

每周例会内容：

		队务赛务，进度汇报，下周规划，技术讲座

会议要求：

		1.严禁迟到，全程参与
	
		2.规定参会人员必须出席，特殊情况必须提前向队长请假
	
		3.提前做好会议准备

----
#### <a name="4">**支出制度**</a>
----
支出流程：

		队员填写物资购买申请表，经指导老师审核并批准后方可支出；
		收到物资后，收集发票，整理提交给指导老师；
支出审核：

		项目管理&运营组 统计财务支出和物资购买信息（频率：每周一次）；
		项目管理&运营组 更新支出流水表至项目管理文档（频率：每月一次）；
----
#### <a name="6">**培训制度**</a>
----

先期培训：

		时间：9月~10月
		
		内容：机器人概况，基本结构，组别分工
		
		目的：培训梯队队员
RM专项培训：

		时间：11月~12月
		
		内容：发射结构等RM专属模块，科学设计方法，人机工程，仿真
		
		目标：培训正式队员
